# Журнал успеваемости студентов

Консольное приложение для учета успеваемости студентов на языке Go.

## Функциональность

- Добавление студентов с оценками
- Просмотр всех студентов с средним баллом
- Фильтрация студентов по среднему баллу
- Валидация вводимых оценок (только от 1 до 5)

## Запуск

```bash
go run journalzad6.go
